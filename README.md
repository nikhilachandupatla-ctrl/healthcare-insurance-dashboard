# healthcare-insurance-dashboard

Excel-based healthcare insurance dashboard analyzing BMI, age, smoking, and other factors using pivot tables, KPIs, and data visualization.

## 📊 Healthcare Insurance Dashboard Project

This project is an Excel-based healthcare insurance analytics dashboard created using data analysis and visualization techniques.

## 🎯 Objective

To analyze factors affecting insurance charges such as BMI, age, smoking, number of children, and region.

## 🧱 Project Structure

- **Sheet 1 and 2:** Clean Data, KPI calculations, and Pivot Tables  
- **Sheet 3:** Data Cleaning Log  
- **Sheet 4:** Dashboard with visualizations and insights  

## 📊 Tools Used

Microsoft Excel (Pivot Tables, Pivot Charts, KPI Cards, Slicers)

## 🔍 Key Insights

- BMI strongly impacts insurance charges  
- Smoking significantly increases cost  
- Age has a major effect on insurance charges  
- Number of children slightly influences charges  
- Regional variation is minimal compared to lifestyle factors  

## 🧠 Skills Demonstrated

Data Cleaning • Data Analysis • Data Visualization • Dashboard Design • KPI Development • Data Storytelling
